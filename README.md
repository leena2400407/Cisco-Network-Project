🌐 Cisco Enterprise Network Project

A comprehensive enterprise network designed and implemented using **Cisco Packet Tracer**. This project demonstrates the configuration of modern networking technologies including VLANs, dynamic routing protocols, DHCP, NAT, VPN, and network management services.



📖 Project Overview

This project simulates the network infrastructure of an enterprise organization with multiple buildings and a branch office. The network is designed to provide secure, scalable, and reliable communication between all departments while supporting various network services.

The implementation includes:

- Multi-building network topology
- VLAN segmentation
- Dynamic routing
- IP addressing and subnetting
- DHCP services
- NAT configuration
- VPN connectivity
- DNS and Email services
- Network management features
